# Bangladesh Height Percentile

**Bangladesh Height Percentile** is an interactive web tool that allows users to see how their height compares to the adult population in Bangladesh. It calculates the percentile and visually shows the result on a bell curve.

## Features

* Input height in **centimeters (cm)** or **feet + inches**.
* Select **gender** (male/female) for accurate comparison.
* Calculate your **height percentile** relative to Bangladeshi adults.
* Visual **bell curve visualization** with gradient fill.
* **Shaded area under the curve** representing your percentile.
* Animated marker showing your height on the curve.
* Detailed explanation of mean, standard deviation, z-score, and percentile.
* Fully **responsive UI** optimized for desktop and mobile.

## How to Use

1. Clone or download the repository.
2. Open the `index.html` file in your web browser.
3. Select your gender.
4. Choose the height input method (cm or ft+in).
5. Enter your height.
6. Click **Calculate Percentile** to see your results.

## Data Sources

* **Male height:** Khadem & Islam 2014, Parash 2011 — Mean = 167.7 cm, SD = 6.43 cm
* **Female height:** BDHS pooled data (JOGH 2023) — Mean = 150.9 cm, SD = 5.6 cm

> Percentile calculations are based on a normal distribution approximation using the z-score method.
